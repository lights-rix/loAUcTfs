# 项目简介

## 核心功能
- 基于 YOLOv8 的垃圾检测系统，支持 24000 张标注数据集（'waste' 类别）
- 支持图片、视频及摄像头进行实时检测
- 可自定义置信度和 IOU 阈值
- 界面实时显示目标位置、目标总数及置信度信息
- 支持批量检测并查看所有结果
- 支持检测结果保存功能
- 提供多种检测模型选择
- 可选择显示标签和原图

## 适用场景
- 无人机环境下的垃圾检测与分类
- 城市环境清洁自动化监测
- 大型区域垃圾分布快速统计
- 环保数据采集与分析

## 技术特点
- 基于 Python 3.9 开发，依赖 OpenCV-Python、PyQt5、Ultralytics、Torch1.9
- 使用 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
- 提供完整的 UI 界面源文件和图标
- 包含完整的程序文件、数据集图片和项目运行教程
